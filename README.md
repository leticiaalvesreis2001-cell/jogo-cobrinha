# 🐍 Snake Game

Um jogo moderno da clássica **Cobrinha (Snake)** desenvolvido utilizando apenas tecnologias nativas do navegador: **HTML5, CSS3 e JavaScript Vanilla**.

O projeto foi construído em um único arquivo HTML, sem dependências externas, oferecendo uma implementação leve, rápida e compatível com os principais navegadores modernos.

## 🧠 Regras do Jogo

1. Controle a cobrinha utilizando as setas do teclado.
2. Consuma os alimentos espalhados pelo mapa.
3. Cada alimento aumenta seu tamanho e sua pontuação.
4. Evite colidir com:

   * As paredes do cenário
   * O próprio corpo
5. Ao colidir, a partida é encerrada.
